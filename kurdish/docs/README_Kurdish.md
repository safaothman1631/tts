# سیستەمی گۆڕینی نووسین بۆ دەنگ بە پشتگیری زمانی کوردی 🎤

سیستەمێکی تەواو بۆ گۆڕینی نووسین بۆ دەنگ لەگەڵ پرۆسێسکردنی زمانی سروشتی، بە پشتگیری زمانی **کوردی** و **ئینگلیزی**.

## تایبەتمەندییەکان ✨

### پرۆسێسکردنی زمانی سروشتی (NLP)
- **ئاساییکردنی دەق**: فراوانکردنی کورتکراوەکان و کاراکتەرە تایبەتەکان
- **گۆڕینی ژمارە بۆ وشە**: ژمارەکان، دراو، بەروار و ڕێژە دەگۆڕێت بۆ وشە
- **ناسینەوەی زمان**: بە شێوەیەکی خۆکار زمانی دەقەکە دەناسێتەوە
- **شیکاری هەست**: هەستەکان دەشیکاتەوە و دەنگی گونجاو پێدەدات
- **بەشکردنی ڕستە**: دەقەکە بۆ ڕستەکانی سروشتی دابەش دەکات

### تایبەتمەندییەکانی TTS
- **چەند بزوێنەرێک**: پشتگیری pyttsx3 (ئۆفلاین) و gTTS (ئۆنلاین)
- **دەنگی گونجاو بە هەست**: خێرایی، قەبارە و بڵێندی بەپێی هەست ڕێک دەخات
- **پشتگیری زمانی کوردی**: بە بەکارهێنانی دەنگی عەرەبی بۆ کوردی
- **هاوردەی چەند دەقێک**: چەندین دەق بە کاراییەوە پرۆسێس دەکات
- **هەناردەی فایلی دەنگ**: دەنگەکە وەک MP3 پاشەکەوت دەکات

## دامەزراندن 📦

```powershell
# چوونە بوخچەی پرۆژە
cd c:\Users\SAFA\tts

# دامەزراندنی پێداویستییەکان
pip install -r requirements.txt

# داگرتنی داتای NLP (خۆکارانە لە کاتی یەکەم جارە ڕادەکات)
python -m textblob.download_corpora
```

## بەکارهێنان 🚀

### بەکارهێنانی هێڵی فەرمان بە کوردی

**بەکارهێنانی بنەڕەتی:**
```powershell
python main.py "سڵاو! بەخێربێیت بۆ سیستەمەکە" --engine gtts
```

**پاشەکەوتکردن لە فایل:**
```powershell
python main.py "دەقی کوردی" --save --output kurdish_audio --engine gtts
```

**لیستی دەنگە بەردەستەکان:**
```powershell
python main.py --voices
```

### شێوازی کۆدنووسی

```python
from tts_pipeline import TTSPipeline

# دروستکردنی pipeline بە gTTS بۆ پشتگیری کوردی
pipeline = TTSPipeline(engine_type="gtts")

# پرۆسێسکردن و قسەکردن
text = "سڵاو! چۆنی؟"
pipeline.process_and_speak(text)

# پرۆسێسکردن و پاشەکەوتکردن
pipeline.process_and_save(
    text="ئەم دەنگە بە کوردی دەپارێزرێت",
    output_filename="kurdish_output",
    use_emotion=True
)

# پرۆسێسکردنی چەند دەقێک
texts = [
    "بەیانی باش",
    "ئێوارەتان باش", 
    "شەوتان خۆش"
]
results = pipeline.batch_process(texts, output_prefix="greetings")
```

## ڕێکخستن ⚙️

دەستکاری `config.py` بکە بۆ گۆڕینی ڕێکخستنەکان:

```python
# ڕێکخستنی بزوێنەری TTS
TTS_ENGINE = "gtts"  # باشترە بۆ کوردی
VOICE_RATE = 150  # وشە لە خولەکدا
VOICE_VOLUME = 0.9  # 0.0 بۆ 1.0

# ڕێکخستنی NLP
DEFAULT_LANGUAGE = "en"
SUPPORTED_LANGUAGES = ["en", "ku", "ar"]  # ئینگلیزی، کوردی، عەرەبی
ENABLE_SENTIMENT = True

# کورتکراوەکانی کوردی
KURDISH_ABBREVIATIONS = {
    "د.": "دکتۆر",
    "پ.": "پرۆفیسۆر",
    "م.": "مامۆستا",
}
```

## چارەسەرکردنی کێشەکان 🔧

**کێشە: کێشەی دانان لە numpy**
- ئێمە کورتکراوەکانمان ساکار کردووە بۆ دوورکەوتنەوە لە کێشەی وەشانکردن

**کێشە: داتای NLTK نەدۆزراوەتەوە**
- بەرنامەکە خۆکار داتاکە دادەگرێت لە یەکەم جار
- بۆ دامەزراندنی دەستی: `python -m textblob.download_corpora`

**کێشە: پشتگیری کوردی**
- بەکاربهێنە `engine_type="gtts"` بۆ پشتگیری باشتری کوردی
- gTTS دەنگی عەرەبی بەکاردەهێنێت بۆ دەقی کوردی

**کێشە: دەرچوونی دەنگ نییە**
- بپشکنە قەبارەی سیستەم و ئامێرەکانی دەنگ
- بەکاربهێنە `--save` بۆ پاشەکەوتکردن لە جیاتی لێدان

## پێداویستییەکان 📋

- Python 3.8+
- سیستەمی Windows
- پەیوەندی ئینتەرنێت (بۆ بزوێنەری gTTS)
- ئامێری دەرچوونی دەنگ

## تێبینییەکانی گرنگ ⚠️

1. **بزوێنەری pyttsx3** لەوانەیە بە باشی پشتگیری کوردی نەکات
2. **بزوێنەری gTTS** پێشنیارە بۆ دەقی کوردی (`engine_type="gtts"`)
3. دەقی کوردی وەک زمانی **فارسی (fa)** دەناسرێتەوە لە langdetect
4. سیستەمەکە دەنگی عەرەبی بەکاردەهێنێت بۆ لێدانی دەقی کوردی
