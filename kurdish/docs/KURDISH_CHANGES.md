## بەکارهێنانی خێرا / Quick Usage

### بە کوردی:
```python
from tts_pipeline import TTSPipeline

pipeline = TTSPipeline(engine_type="gtts")
pipeline.process_and_speak("سڵاو! بەخێربێیت")
```

### لە هێڵی فەرمان:
```powershell
# تاقیکردنەوە
python test_kurdish.py

# ئەپی کوردی
python kurdish_app.py

# نیشاندانی دوو زمانی
python demo_bilingual.py
```

## تێبینییەکانی تەکنیکی / Technical Notes

1. **ناسینەوەی زمان**: 
   - کوردی وەک "fa" (فارسی) دەناسرێتەوە لە langdetect
   - ئەمە ئاسایییە لەبەر نزیکی نووسینی کوردی و فارسی

2. **دەنگی TTS**:
   - pyttsx3: لەوانەیە باش کار نەکات بۆ کوردی
   - gTTS: پێشنیارە - دەنگی عەرەبی بەکاردەهێنێت

3. **کۆدکردنی کاراکتەر**:
   - هەموو فایلەکان UTF-8 بەکاردەهێنن
   - پشتگیری تەواوی کاراکتەرە کوردییەکان
