# Kurdish TTS

پڕۆژەی دەنگاندنی نووسین بە کوردی.

## پێکهاتە

- `apps/` — ئەپلیکەیشنە جیاجیاکانی کوردی (offline / simple / examples)
- `training/` — سکریپتەکانی ڕاهێنان و تاقیکردنەوە
- `tools/` — تۆڵە کۆمەکیەکان: voice recorder, phonetic readers, fixes
- `docs/` — ڕێبەرەکان (HOW_TO_USE, QUICKSTART, VOICE_GUIDE)
- `data/` — داتاسێت، تۆمارەکان، مۆدێلی ڕاهێنراو
